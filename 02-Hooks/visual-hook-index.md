---

## Selection Algorithm

When a skill needs a visual hook for video content:

1. **FILTER** by Platform (TikTok/Instagram/YouTube/All) AND Content Type (talking head/b-roll/screen record/mixed)
2. **EXCLUDE** any visual hook where `Last Used` is within 7 days
3. **SORT** by Score (highest first)
4. **PICK top 3** and select the one that best matches the topic and emotion of the content
5. **COMBINE** with a text hook from hook-index.md (visual hook = what they SEE, text hook = what they HEAR/READ)
6. **UPDATE** `Last Used` to today

### Rotation Rule

A visual hook with `Last Used` within 7 days is **blocked** from selection. This forces visual variety. If ALL visual hooks in a filter are blocked, expand to adjacent Content Types or fall back to TEMPLATE tier.

### Promotion Path

- Use a STRONG or UNTESTED visual hook → track in 07-Analytics
- If the video outperforms 30-day avg by 20%+ → promote to SELF-PROVEN (score 85+)
- If it underperforms twice → demote by 10 points

---

## Scoring Rubric (4 Criteria x 5 Points = /20, normalized to 0-100)

| Criterion | 1 (Weak) | 3 (Average) | 5 (Strong) |
|-----------|----------|-------------|------------|
| **STOP POWER** | Viewer might scroll past | Noticeable but not arresting | Freezes a fast-scrolling thumb in under 0.5 seconds |
| **CURIOSITY VISUAL** | Scene is self-explanatory | Mild "what is that?" | Creates a visual question that demands answers |
| **UNIQUENESS** | Standard talking-head-in-front-of-mic | Slight twist on common setup | Completely breaks expected feed patterns |
| **EXECUTION EASE** | Needs crew/props/location/VFX | Doable solo with some prep | Phone + tripod, shoot in 5 minutes |

# 09-Skills: AI Team Skills Index

## What This Folder Contains
Each file in this folder is a skill that teaches Claude Code (or Claude AI) how to perform a specific team role. Read the relevant skill file before executing any task.

## How to Use
**Slash commands (recommended):** Type the command directly in Claude Code:
- `/ghostwriter write 5 tweets about AI agents in DeFi`
- `/content-strategist plan my content this week`
- `/data-analyst weekly analytics`

**Manual invocation:** Say "Read 09-Skills/[skill-name].md and [your request]"

## Niche Knowledge

## What AI Can Fully Replace
- Operations Lead (task management, reporting)
- Content Strategist (planning, calendars)
- Ghostwriter (all written content)
- Data Analyst (tracking, analysis, reporting)
- Email/CRM copy (sequences, follow-ups)
- Funnel copy (landing pages, lead magnets)

## What AI Partially Helps (Human Still Needed)
- Video Editor: AI writes scripts and direction. Human edits the footage.
- Funnel Builder: AI writes copy and structure. Human builds in Framer/Webflow.
- Community Manager: AI writes all copy. Human sends messages and handles live interaction.
- Sales Closer: AI writes scripts and objection banks. Human has the actual conversations.

## Skill Chaining (Using Multiple Skills Together)
Common workflows that use multiple skills in sequence:

**Weekly Content Production:**
1. content-strategist.md (plan the week)
2. ghostwriter.md (write the posts)
3. video-editor.md (script the videos)
4. operations-lead.md (create the task list for execution)

**Launch Campaign:**
1. funnel-builder-email-crm.md (build the landing page and email sequence)
2. content-strategist.md (plan the launch content)
3. ghostwriter.md (write the launch posts)
4. community-manager.md (write the community announcements)
5. sales-closer.md (write the DM outreach scripts)

**Monthly Review:**
1. data-analyst.md (generate the monthly report)
2. content-strategist.md (adjust next month's strategy based on data)
3. operations-lead.md (create next month's task priorities)

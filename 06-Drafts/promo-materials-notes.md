## Post Adaptation Rules (before posting any copy from this PDF)

1. No hashtags on X/Twitter — remove any that appear in draft copy
2. No em dashes — use plain dashes or restructure the sentence
3. Replace generic AI tone — check every post against voice-rules.md
4. "DM me COURSE" / "DM COURSE" — confirm the correct keyword is set up in ManyChat before using
5. Platform tone adaptation:
   - X: Short, punchy, conversation-starter energy
   - Telegram: Warmer, community-facing, "Krib members" language
   - Instagram: Visual hook referenced in caption — first line must make sense without seeing the image
   - LinkedIn: Professional framing — lead with outcome, not hype

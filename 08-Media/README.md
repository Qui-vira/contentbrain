# Media Uploads

Drop your media files here for posting.

## Structure
- `images/` — Screenshots, graphics, memes, carousel slides
- `videos/` — Recorded clips, edited videos, screen recordings

## How it works
When you tell /post "post this with an image" or "post this with a video", it will ask you:
1. Use the latest file from /video-editor output (Desktop/content-studio/out/)
2. Let you specify a file path

If you drop files here, just tell /post: "use the image in 08-Media/images/filename.png"

# Graphic Design Principles for Social Media Content Creation

> Research compiled 2026-03-23. Actionable rules extracted from 15+ professional sources.

---

## 1. Typography System Rules

### Font Selection
- **Maximum 2 font families per design** — one for headlines, one for body text
- Use **sans-serif fonts** for primary text (especially on dark backgrounds — cleaner, more readable)
- Reserve decorative/serif fonts for accent elements only
- **Avoid thin typefaces entirely** — they disappear on mobile screens
- Bold, wide typefaces ensure readability at a glance

### Font Pairing Strategy
- Pair a **serif headline** with a **sans-serif body** (or vice versa)
- If using fonts from the same family, **skip a weight** to create contrast (e.g., Light + Bold, not Regular + Medium)
- Never pair two decorative fonts together

### Type Scale Ratios
Use a mathematical ratio to size your text hierarchy. Pick ONE ratio and apply it consistently:

| Ratio | Name | Best For |
|-------|------|----------|
| 1.25 | Major Third | Gentle hierarchy, lots of text |
| 1.414 | Augmented Fourth | Balanced readability + differentiation |
| 1.618 | Golden Ratio | Maximum aesthetic harmony |

**Golden Ratio example (base 16px):**
- Body text: 16px
- Subheading: 26px (16 x 1.618)
- Heading 2: 42px (26 x 1.618)
- Heading 1: 68px (42 x 1.618)

### Minimum Font Sizes for Mobile
- **Headlines: 32-48pt**
- **Body text: 18-24pt**
- **Captions: 16pt minimum**
- Fine print: avoid entirely on social media

### Line Spacing (Leading)
- Leading should be **~130% of character height** (30% more than the font size)
- Example: 16px font = 21px line height

### Text Coverage Rule
- **No single slide should exceed 20% text coverage** — the rest should be visual
- Think "flashcard" not "blog paragraph"
- One headline + one subhead maximum per carousel slide

## 2. Color Theory for Dark Themes

### Background Colors
- **Never use pure black (#000000)** — causes eye strain and harsh contrast
- Use **dark gray #121212** (Material Design recommendation) as your base surface color
- Dark gray allows a wider range of depth and elevation expression
- White text on dark gray has lower contrast than white on black = easier on eyes

### The 60-30-10 Color Rule
Divide your color usage into three proportions:
- **60% — Dominant color** (dark background: #121212 or similar dark neutral)
- **30% — Secondary color** (supporting color for cards, sections, secondary elements)
- **10% — Accent color** (CTAs, highlights, emphasis — this is where your brand pop lives)

For Quivira brand: 60% dark background, 30% dark gray/charcoal elements, 10% red accents

### Contrast Ratio Requirements
| Element | Minimum Ratio | Standard |
|---------|--------------|----------|
| Normal body text | **4.5:1** | WCAG AA |
| Large text (24px+ or 18.5px bold) | **3:1** | WCAG AA |
| Enhanced readability text | **7:1** | WCAG AAA |
| White text on #121212 | **15.8:1** | Exceeds AAA |

### Saturation Rules for Dark Mode
- **Reduce saturation by ~20 points** compared to light mode versions
- Saturated colors fail WCAG contrast tests on dark backgrounds
- Desaturated colors pass AA standard (4.5:1) consistently
- Use fully saturated brand colors **sparingly** — only on logos, branded buttons, key accent moments

### Dark Mode Elevation System
- Higher elements = lighter surface colors (add white overlay)
- Lower/background elements = darker surface colors
- Use **12% white overlay** on interactive elements (buttons, cards) for state contrast
- Never simply invert a light design — it breaks hierarchy and readability

### Accessible Color Combinations on Dark
- Navy blue + white
- Dark gray + cream
- Purple + light gray
- Black + yellow (high visibility)
- **Avoid red-green combinations** (8% of men are red-green colorblind)

## 3. Visual Hierarchy Principles

### The Hierarchy Stack (Order of Visual Weight)
1. **Size** — Largest elements are read first. Headlines dominate.
2. **Color/Contrast** — High-contrast elements grab attention next.
3. **Position** — Top-left and center elements are seen before bottom-right.
4. **White space** — Isolated elements with breathing room appear more important.
5. **Typography weight** — Bold text outranks regular weight.

### Scanning Patterns
- **Z-Pattern**: For visual-heavy designs with minimal text (ads, product showcases). Eye moves: top-left → top-right → bottom-left → bottom-right.
- **F-Pattern**: For text-heavy posts (educational lists, quote cards). Eye scans: top line → drops down left side → scans partial lines.

### Color Temperature for Hierarchy
- **Warm colors (red, orange, yellow)** demand more attention — use for primary elements
- **Cool colors (blue, green, purple)** recede — use for supporting/background elements
- To make ONE element pop: use warm color against cool background

### White Space Rules
- White space is not wasted space — it directs focus
- Surround your key message with generous space to make it the focal point
- Cluttered designs cause **43% higher scroll-past rate**

### The "One" Rule
Every design should have:
- **One message**
- **One emotion**
- **One focal point**
- **One goal**

Multiple competing focal points overwhelm viewers. A gym promotion reducing from 5 offers to 1 saw a **156% increase in click-through rate**.

## 4. Layout & Composition Rules

### Platform Dimensions (Critical Reference)
| Platform | Post Type | Dimensions |
|----------|-----------|------------|
| Instagram Post (square) | Feed | 1080 x 1080px |
| Instagram Post (portrait) | Feed | 1080 x 1350px (4:5) |
| Instagram Story/Reel | Vertical | 1080 x 1920px |
| LinkedIn Post | Feed | 1200 x 627px |
| Twitter/X Post | Feed | 1600 x 900px |
| Facebook Post | Feed | 1200 x 630px |
| TikTok | Vertical | 1080 x 1920px |

### Instagram Story/Reel Safe Zones
- **Avoid top 250px** (profile bar, notifications)
- **Avoid bottom 250px** (CTA buttons, swipe-up area)
- Safe content area: center 1080 x 1420px

### Rule of Thirds
- Divide your canvas into a 3x3 grid
- Place your primary focal point at one of the four intersection points
- Never center everything — off-center placement creates visual energy

### Mobile Tap Targets
- Minimum button/CTA size: **44 x 44 pixels**
- Design vertical-first — horizontal layouts break on mobile
- Center-align key text for mobile consumption

### Image Quality Standards
- Minimum resolution: **1080px wide**
- JPG compression: 80-85% quality
- Use **JPG for photos**, **PNG for text-heavy graphics**
- Color profile: **sRGB** for web/social
- Target file size: under 1MB per image

## 5. Carousel-Specific Design Patterns

### Technical Specs
- Recommended size: **1080 x 1350px** (4:5 portrait) — maximum screen real estate
- Sweet spot: **8-10 slides** for engagement (6-13 acceptable range)
- Maximum: 20 slides per carousel (Instagram 2026)
- Per-slide file size: under 30MB
- Mixed media (images + video clips) get **2.33% engagement** but only 7% of carousels use this

### Carousel Performance Data
- Carousels achieve **1.92% average engagement** (vs. 0.50% Reels, 0.45% static images)
- **1.4x wider reach** and **3.1x higher engagement** vs. single photos
- **95% higher save rate** than other formats
- **27% longer session duration** than single images
- If someone doesn't swipe, Instagram will **auto-advance to slide 2** as a second chance

### Slide Structure Framework

**Slide 1 — The Hook:**
- Maximum 8-10 words for headline
- Must stop scrolling and communicate immediate value
- Make it feel INCOMPLETE — cropped diagrams, partial checklists, blurred results
- Bold typography + high contrast + strategic white space

**Slide 2 — Reduce Skepticism:**
- Preview the structure of what's coming
- Establish credibility immediately
- "Here's what you'll learn" or "The 5 steps"

**Slides 3-8 — The Value:**
- ONE point per slide (flashcard format)
- Consistent layout with changing accent element (visual rhythm)
- Use open loops: "Next: the fix that takes 2 minutes"

**Slide 7-9 — Proof:**
- Data, testimonials, screenshots, visual proof
- Numbers and statistics anchor credibility

**Second-to-Last Slide — Soft CTA:**
- "Save this for later" or "Tag someone who needs this"

**Final Slide — Hard CTA:**
- Explicit call-to-action: follow, save, share, comment, link in bio
- Two CTAs total per carousel: one mid-way, one at the end

### Seamless Carousel Technique (Canva)
- Create one wide canvas: **1350px high x (1080px x number of slides)**
- Example: 6-slide carousel = 1350 x 6480px
- Add vertical guides at every 1080px interval (1080, 2160, 3240, 4320, 5400)
- Place interesting elements at the slide intersections to encourage swiping
- Export and slice into individual slides

### Carousel Content Formats That Work
- Listicles (numbered tips)
- Before/after transformations
- Step-by-step tutorials
- Myth vs. reality
- Decision trees and checklists
- Data breakdowns with one stat per slide

### Template Strategy
- Build **3-5 reusable templates** so followers recognize your series
- Pick one weekly carousel format for consistency
- Create evergreen "saver" content: templates, checklists, pricing ranges

## 6. Scroll-Stopping Techniques (Psychology)

### The 13-Millisecond Rule
The human brain processes and recognizes an image in **13 milliseconds** — faster than reading a single word. Your visual must communicate BEFORE the text is read.

### The 3-Second Window
You have **under 3 seconds** to stop the scroll. The opening frame of any content must be the most visually striking element.

### The 50-Millisecond Judgment
Users form first impressions in **~50 milliseconds** and these judgments typically remain unchanged even with extended exposure. First impression IS the final impression.

### Pattern Breaking
The brain ignores expected patterns and notices disruption. Your content must create a "pattern break" — something that stands out from the uniform feed:
- Unexpected color in a sea of similar tones
- Bold text against clean negative space
- Human face with exaggerated emotion
- Asymmetrical layout when everything else is centered

### Face Psychology
- Human faces are processed faster than any other visual information (evolutionary wiring)
- Thumbnails with clear, expressive faces get **up to 20% higher click-through rates**
- **Direct eye contact** amplifies personal engagement
- Most effective facial expressions: genuine surprise, curiosity/confusion, excitement

### High Contrast = High Attention
- High-contrast thumbnails improve click-through rates by **20-40%**
- Critical for mobile where details disappear at small sizes
- Bright foreground colors against dark/neutral backgrounds = maximum contrast
- Bold separation between subject and background ensures readability at any size

### Curiosity Gap
- Hint at information without revealing it completely
- The brain hates incomplete information and will pause to resolve uncertainty
- "You're missing something important" creates urgency
- Open loops: "These 7 mistakes..." or "The #1 reason..."
- Ethical curiosity outperforms misleading clickbait long-term

### Emotional Triggers That Stop Scrolling
1. **Surprise/Unexpectedness** — unexpected outcomes or twists
2. **FOMO** — time-sensitive or exclusive messaging
3. **Empathy/Relatability** — shared struggles, "this is me" moments
4. **Joy/Nostalgia** — warm emotional associations
5. **Curiosity** — unanswered questions, incomplete information

### Micro-Narrative Frameworks
Structure your visual story using one of these:
- **Before → After** (transformation)
- **Problem → Struggle → Fix** (solution journey)
- **Unexpected Outcome** (surprising resolution)

### Simplicity Wins
- Communicate ONE clear idea per thumbnail/cover
- Busy designs cause cognitive overload → instant scroll-past
- "Thumbnails that reduce uncertainty perform better"
- Clarity beats cleverness in every test

## 7. Common Mistakes to Avoid

### The Deadly 10

| # | Mistake | Impact | Fix |
|---|---------|--------|-----|
| 1 | Wrong image dimensions per platform | 67% lower organic reach, 52% fewer clicks | Use platform-specific dimensions (see Section 4) |
| 2 | Unreadable text on mobile | 43% higher scroll-past rate | Minimum 32pt headlines, 18pt body, 4.5:1 contrast |
| 3 | Inconsistent branding | 67% lower brand recall | Define 2-3 colors, 2 fonts, logo rules in a brand kit |
| 4 | Overcrowded designs | Viewers scroll past immediately | One message, one focal point, generous white space |
| 5 | Poor color accessibility | Excludes 8% of male viewers | Avoid red-green combos, test with colorblind simulators |
| 6 | Low-quality/pixelated images | Destroys credibility | Minimum 1080px wide, 80-85% JPG quality, sRGB |
| 7 | Ignoring mobile optimization | Miss 89% of audience | Design vertical-first, test on phone before posting |
| 8 | Missing or weak CTAs | Wastes conversion opportunity | Contrasting color CTA, bottom-right placement, clear action |
| 9 | Inconsistent visual style | Damages professional perception | Document 3-5 style keywords, create templates |
| 10 | One-size-fits-all across platforms | Fails on every platform | Adapt design for each platform's audience and algorithm |

### Additional Mistakes
- **Using pure black (#000000) backgrounds** — causes eye strain; use #121212
- **More than 2 fonts** — creates visual chaos
- **Saturated colors on dark backgrounds** — fails contrast tests; desaturate by 20 points
- **Designing desktop-first** — 89% of social media is consumed on mobile
- **Text-heavy slides** — exceed 20% text coverage and engagement drops
- **No visual rhythm in carousels** — slides feel disconnected; use consistent templates
- **Skipping the hook** — slide 1 must stop the scroll; everything else is irrelevant without it
- **Horizontal layouts** — mobile screens are vertical; stack content vertically
- **Overdesigning** — more effects ≠ more impact; clarity always wins
- **Ignoring safe zones** — key content hidden behind platform UI elements

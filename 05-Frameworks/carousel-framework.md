# Carousel Framework

## Specs
- Size: 1080x1350 pixels (4:5 ratio)
- Slide count: 7-10 slides (sweet spot for engagement; 10-slide carousels maximize dwell time)
- Style: dark background, clean typography, red accents (Quivira brand)
- Max 20 slides allowed on Instagram

## Slide Structure

| Slide | Purpose | Content | Overlay Opacity |
|-------|---------|---------|-----------------|
| 1 | Cover | Hook headline (<12 words). Must stop the scroll. "SWIPE >" at bottom. | 50-55% (lightest) |
| 2 | Context | The "what" and "why" in 2-3 sentences. Why this matters to the reader. | 60% |
| 3-7 | Steps | Core content. One idea per slide. Checklist or numbered format. 3-5 items per slide. | 65-70% (readability) |
| 8 | Proof | Outcome stat in GOLD, result, or testimonial. Glass panel treatment. | 65% |
| 9 | CTA | "Comment [KEYWORD] and I will DM you the full guide." Large button with drop shadow. | 50% (second lightest) |

## Narrative Structures (pick one per carousel)

| Structure | When to Use | Flow |
|-----------|-------------|------|
| Problem > Solution > Proof | Educational, value content | Hook problem > Explain > Steps > Result > CTA |
| Step-by-Step Tutorial | How-to, process content | Hook > Step 1 > Step 2 > ... > Result > CTA |
| Before > Process > After | Transformation content | Before state > What changed > How > After > CTA |
| Myth vs. Fact | Contrarian, authority content | Hook myth > Facts > Truth > Action > CTA |
| Data Visualization | Market analysis, stats | Hook stat > Context > Data > Data > Insight > CTA |
| Listicle | Tips, resources | Hook > Item 1 > Item 2 > ... > Summary > CTA |

## Typography Hierarchy

| Element | Weight | Size | Glow? | Letter Spacing |
|---------|--------|------|-------|---------------|
| Hero word/number | Black (900) | 72-80pt | YES (14px blur) | 0 to +50 |
| Main headline | Bold (700) | 52-60pt | YES (10px blur) | 0 |
| Subheadline | Bold (700) | 36-42pt | Optional (8px) | 0 |
| Body text | Regular (400) | 28-32pt | NEVER | 0 |
| Body emphasis | Bold (700) | 28-32pt | NEVER | 0 |
| Caption/meta | Regular (400) | 22-24pt | NEVER | 0 |
| Badge/tag | Bold (700) | 20-22pt | NEVER | +50 to +100 |
| Slide counter | Regular (400) | 20-22pt | NEVER | +50 |

**Rules:**
- Max 2 font families (1 headline + 1 body)
- Max 3 font sizes per slide
- Minimum 24px for any text
- Headlines get glow. Body text NEVER gets glow.
- Left-align body text always. Center-align only for 1-2 line headlines.

## Layout Rules
- **Margins**: 80px left/right minimum
- **Top safe zone**: 120px (IG profile overlay area)
- **Bottom safe zone**: 80px (branding area)
- **Section gap**: 40-60px between major blocks
- **Line gap**: 10-14px between text lines
- **Card padding**: 20-30px inside glass panels
- **Slide counter**: Top-right corner, slides 2 through N-1 (not on cover or CTA)
- **Branding**: Bottom-left corner, @big_quiv, consistent on all slides

## Depth Requirements (minimum per slide)
Every slide must have at least 3 of these:
1. Dark overlay on AI background
2. Vignette (darken edges)
3. Gradient orb (ambient color depth)
4. Glass panel (semi-transparent card)
5. Text glow (on headlines)
6. Drop shadow (on buttons/panels)
7. Accent line (red separator with glow)

## Hook Text Rules (Slide 1)
- Fewer than 12 words
- Create a curiosity gap requiring swipe to resolve
- Use specific numbers ("16 Tokens" not "Some Tokens")
- Include time-based urgency when possible
- Emotionally resonant personal transformation language

## Swipe Cues
- "Swipe >" text in GRAY at bottom of cover slide
- Arrow graphics pointing right
- Slide counter (3/8) on content slides
- Partially visible elements at edge implying continuation

## Checklist Slide Rules
- Each item must be a direct instruction the viewer can follow
- Good: "Go to code.visualstudio.com" "Click the Extensions icon" "Search for Claude Code"
- Bad: "Install the tool" "Set things up" "Get started"
- Use checkboxes or numbered items
- One action per line
- No paragraphs, no long explanations

## Caption
- 60-70 words / 400-450 characters
- Complement carousel content, don't repeat slide text
- Add context, personal story, or reinforce CTA
- Include relevant hashtags (IG only)
- Pin a comment with secondary CTA

## Quality Checklist
1. Readable at phone size (375px width)?
2. ONE dominant element per slide?
3. Same fonts/colors/spacing across all slides?
4. Glow on headlines, NOT body text?
5. 80px+ margins on all sides?
6. No AI-generated text in backgrounds?
7. At least 3 depth elements per slide?
8. Slide counter on slides 2 through N-1?
9. Cover has lightest overlay, content has heaviest?
10. CTA button has drop shadow?

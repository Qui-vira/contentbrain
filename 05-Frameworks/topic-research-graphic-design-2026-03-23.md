## Framework 1: The Design Decision Tree

When the /graphic-designer skill receives a request, it should ask these questions IN ORDER to make design decisions:

### Question 1: What is the content type?
| Content Type | Canvas Size | Slide Count | Overlay Range |
|-------------|-------------|-------------|---------------|
| Instagram Carousel | 1080x1350 (4:5) | 5-10 slides | 50-70% |
| Instagram Single Post | 1080x1350 (4:5) | 1 slide | 45-55% |
| YouTube Thumbnail | 1280x720 (16:9) | 1 image | 30-45% |
| TikTok Cover | 1080x1920 (9:16) | 1 image | 40-55% |
| X/Twitter Image | 1200x675 (16:9) | 1 image | 40-55% |
| LinkedIn Carousel | 1080x1350 (4:5) | 5-10 slides | 55-70% |

### Question 2: What is the goal?
| Goal | Design Treatment |
|------|-----------------|
| Stop the scroll | Maximum contrast, face or eye-catching element, bold single statement |
| Teach/Educate | Clean hierarchy, numbered steps, glass panels for structure |
| Build authority | Data visualization, stats in gold, professional spacing |
| Drive saves | Dense value per slide, checklist format, reference-worthy design |
| Drive comments | Question in headline, provocative visual, open loop |
| Sell/Convert | CTA button with drop shadow, urgency colors (red), benefit-focused headline |

### Question 3: What mood/energy?
| Mood | Background Style | Lighting | Accent Colors |
|------|-----------------|----------|---------------|
| Powerful/Bold | Dramatic, cosmic, cinematic | Side light, high contrast | Red, Gold |
| Educational/Clean | Minimal, subtle gradients | Even, soft | Blue, White |
| Urgent/Breaking | Dark with red accents | Spotlight, harsh | Red, Orange |
| Premium/Luxury | Dark with gold elements | Warm, golden hour | Gold, Cream |
| Tech/Futuristic | Neon, circuit, cyber | Neon glow, screen light | Cyan, Purple |
| Community/Warm | Warm tones, human elements | Golden, practical | Amber, Warm white |

## Framework 2: The Typography Decision Matrix

Instead of guessing fonts, use this matrix:

| Slide Element | Font Weight | Size Range | Glow? | Letter Spacing |
|--------------|-------------|------------|-------|---------------|
| Hero number/word | Black/Heavy (900) | 72-80pt | YES (14px radius) | 0 to +50 |
| Main headline | Bold (700) | 52-60pt | YES (10px radius) | 0 |
| Subheadline | Bold (700) | 36-42pt | Optional (8px) | 0 |
| Body text | Regular (400) | 28-32pt | NEVER | 0 |
| Body emphasis | Bold (700) | 28-32pt | NEVER | 0 |
| Caption/meta | Regular (400) | 22-24pt | NEVER | 0 |
| Badge/tag | Bold (700) | 20-22pt | NEVER | +50 to +100 |
| Slide counter | Regular (400) | 20-22pt | NEVER | +50 |

## Framework 3: The Depth Layer Stack

Every premium slide should have these layers (bottom to top):

```
Layer 1: AI-generated background (1080x1350)
Layer 2: Dark overlay (variable opacity by slide type)
Layer 3: Vignette (darken edges, lighten center)
Layer 4: Gradient orbs (ambient color depth, 1-2 per slide)
Layer 5: Glass panels (for content cards, semi-transparent)
Layer 6: Text glow layer (blurred colored text behind headlines)
Layer 7: Crisp text (headlines, body, tags)
Layer 8: Accent elements (lines, badges, counters, buttons)
Layer 9: Branding (watermark, @handle, slide counter)
```

**Minimum depth requirement**: Every slide must have at least 3 of these depth layers beyond the basic background + text.

## Framework 5: Smart Questioning Protocol

When /graphic-designer receives a design request, it should gather context by asking:

**Essential questions (always ask):**
1. "What content are we designing? (carousel, single post, thumbnail, cover)"
2. "What's the goal? (reach, saves, sales, authority)"
3. "Is the text/copy ready, or should I draft it too?"

**Contextual questions (ask when relevant):**
4. "Any specific mood? (bold, clean, premium, urgent, tech)" — only if not obvious from content
5. "Is this part of a series or standalone?" — affects branding consistency
6. "Any specific images or references to match?" — only if user hints at visual reference

**Never ask:**
- Color choices (always use Quivira brand palette)
- Font choices (always use the typography matrix above)
- Spacing (always use the standard spacing rules)
- Technical specs (always use 1080x1350 for IG carousels)

## Framework 6: Carousel Narrative Structures

| Structure | When to Use | Slide Flow |
|-----------|-------------|------------|
| Problem > Solution > Proof | Educational, value content | Hook problem > Explain > Steps > Result > CTA |
| Step-by-Step Tutorial | How-to, process content | Hook > Step 1 > Step 2 > ... > Result > CTA |
| Before > Process > After | Transformation content | Hook (before state) > What changed > How > After > CTA |
| Myth vs. Fact | Contrarian, authority content | Hook myth > Facts > Truth > What to do > CTA |
| Data Visualization | Market analysis, stats | Hook stat > Context > Data 1 > Data 2 > Insight > CTA |
| Listicle | Resource sharing, tips | Hook > Item 1 > Item 2 > ... > Summary > CTA |

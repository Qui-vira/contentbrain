# Signal Output Template

> Standard format for all signals generated by /technical-analyst.

---

## Signal Template

```
SIGNAL
---
Pair: [PAIR]
Direction: [LONG/SHORT]
Timeframe: [TIMEFRAME]
Entry: $[PRICE]
Stop Loss: $[PRICE] ([DISTANCE %])
TP1: $[PRICE] ([GAIN %]) [R:R]
TP2: $[PRICE] ([GAIN %]) [R:R]
TP3: $[PRICE] ([GAIN %]) [R:R]
Risk: 1% of capital
Confluence:
- [Confluence 1]
- [Confluence 2]
- [Confluence 3]
- [Confluence N]
Confidence: [LOW/MEDIUM/HIGH] ([X/8] confluences)
---
```

## Platform Formatting

### Telegram (Hustler's Krib)
- Use the signal template as-is with emoji markers
- Add: "Reply ENTERED when you take this trade"

### X/Twitter
- Signal + 1-2 sentence educational context
- Thread format: Signal → Why this setup → ICT concept explanation

### LinkedIn
- Market analysis framing, not raw signal
- "Here's what I'm watching on [PAIR] and why the structure is interesting..."

### TikTok/Instagram
- Chart screenshot with annotations
- 30-second voiceover explaining the setup

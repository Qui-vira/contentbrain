# Production Checklist

## Pre-Production
- [ ] Content plan created and approved
- [ ] Scripts written for all formats
- [ ] Hooks selected (one per format)
- [ ] Screen recordings needed: list and record
- [ ] Green screen background visuals prepared
- [ ] Carousel slides designed or content mapped
- [ ] Freebie/lead magnet created (if applicable)
- [ ] ManyChat or DM automation set up (if applicable)
- [ ] Captions pre-written for all platforms

## Production (Shoot Day)
- [ ] Film studio reel (tripod, lighting, external mic)
- [ ] Film cell phone reel (2-3 takes, pick best)
- [ ] Film green screen reel (check chroma key before full take)
- [ ] Capture all B-roll shots from shot list
- [ ] Record screen recordings if not done in pre-production

## Post-Production
- [ ] Edit studio reel (cuts, B-roll, text overlays, music)
- [ ] Add captions to cell phone reel
- [ ] Edit green screen reel (chroma key, background, text)
- [ ] Render carousel slides as images
- [ ] Export all videos in correct aspect ratios (9:16 for Reels/TikTok/Shorts, 16:9 for YouTube)
- [ ] Review all outputs for quality

## Publishing
- [ ] Day 1: Post cell phone reel (highest organic reach)
- [ ] Day 3: Post green screen reel
- [ ] Day 5: Post tutorial walkthrough or screen recording
- [ ] Day 7: Post studio reel
- [ ] Day 9: Post carousel
- [ ] Cross-post to all platforms per repurposing matrix
- [ ] Sync all posts to Notion content calendar

## Post-Launch Monitoring
- [ ] Check ManyChat/DM delivery is working
- [ ] Track metrics at 24h and 48h marks
- [ ] Log engagement to 07-Analytics/
- [ ] Note top performer for future reference

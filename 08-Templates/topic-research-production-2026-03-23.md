## 2. Pacing Patterns

### YouTube Long-Form Pacing (from transcript analysis of top 10)

| Video Section | Pacing | Visual Change Rate | Technique |
|--------------|--------|-------------------|-----------|
| 0-30 sec (Hook) | Fast | Every 2-3 seconds | Jump cuts, text overlays, B-roll montage |
| 30-90 sec (Setup) | Medium | Every 5-7 seconds | Talking head + graphics alternating |
| 90 sec - midpoint | Slower | Every 8-12 seconds | Deep dive on each point, screen recordings |
| Midpoint | RESET | Pattern interrupt | New visual style, new music, or direct address to camera |
| Post-midpoint | Medium-fast | Every 5-8 seconds | Examples with frequent cuts |
| Last 2 min | Fast | Every 3-5 seconds | Recap montage, CTA, energy boost |

**Key finding from transcripts:** Kallaway and Modern Millie both use a "mid-video reset" -- around the 40-60% mark, they change the visual style, energy level, or address the camera directly. This combats the natural drop-off point. Charisma on Command uses a different technique: interspersing the main content with rapid B-roll analysis segments every 90-120 seconds.

### TikTok Short-Form Pacing (from top 20 videos)

| Duration Bucket | Cut Rate | Text Appearance | Sound Events |
|----------------|----------|-----------------|-------------|
| Under 10 sec | Every 1-2 sec | 2-3 text slams | Sound effect every 2-3 sec |
| 10-20 sec | Every 2-3 sec | 4-6 text labels | Transition sound per example |
| 20-40 sec | Every 3-5 sec | Background text + voiceover | Music bed + accent sounds |
| 40-60 sec | Every 4-6 sec | Minimal text, voiceover-led | Music + 2-3 accent sounds |

**Key finding:** The highest-performing TikToks (3M+ plays) are under 15 seconds. At this duration, every frame must earn its place. The 7-13 second range has the highest completion rate, which TikTok's algorithm rewards disproportionately. @ugc.withrach's 8-second video hit 2.4M plays -- proof that shorter is better for demonstrations.

### Template D: AI Visual Hook Production Workflow
```
1. Record base video (trading breakdown, chart analysis, etc.)
2. Export the last frame as a static image
3. Upload to Nano Banana (via Gemini): "Make this [chart/product/screen] [explode/transform/dissolve/shatter] in a cinematic way"
4. Generate 3-5 variations, pick the best
5. Add AI sound effects via ElevenLabs to match the visual
6. Composite: AI hook (3 sec) + original video in CapCut
7. Add bold text overlay in first frame
8. Export and post
Total production time: ~15-20 minutes per hook
```

## 7. Graphic Design Production Techniques (Added 2026-03-23)

### Pillow (Python) Advanced Text Effects

#### Text Shadow (Drop Shadow)
```python
def draw_text_shadow(img, text, x, y, font, text_color, shadow_color=(0,0,0), offset=(4,4), blur=6):
    shadow = Image.new("RGBA", img.size, (0, 0, 0, 0))
    sd = ImageDraw.Draw(shadow)
    sd.text((x + offset[0], y + offset[1]), text, fill=(*shadow_color[:3], 120), font=font)
    shadow = shadow.filter(ImageFilter.GaussianBlur(blur))
    img = Image.alpha_composite(img.convert("RGBA"), shadow)
    ImageDraw.Draw(img).text((x, y), text, fill=text_color, font=font)
    return img
```

#### Neon Glow (Multi-Layer Stack)
```python
def draw_neon_text(img, text, x, y, font, glow_color, fill_color=(255,255,255)):
    # Layer 1: Wide glow (blur 14)
    glow1 = Image.new("RGBA", img.size, (0, 0, 0, 0))
    ImageDraw.Draw(glow1).text((x, y), text, fill=(*glow_color[:3], 80), font=font)
    glow1 = glow1.filter(ImageFilter.GaussianBlur(14))
    img = Image.alpha_composite(img.convert("RGBA"), glow1)
    # Layer 2: Medium glow (blur 8)
    glow2 = Image.new("RGBA", img.size, (0, 0, 0, 0))
    ImageDraw.Draw(glow2).text((x, y), text, fill=(*glow_color[:3], 120), font=font)
    glow2 = glow2.filter(ImageFilter.GaussianBlur(8))
    img = Image.alpha_composite(img, glow2)
    # Layer 3: Tight glow (blur 3)
    glow3 = Image.new("RGBA", img.size, (0, 0, 0, 0))
    ImageDraw.Draw(glow3).text((x, y), text, fill=(*glow_color[:3], 160), font=font)
    glow3 = glow3.filter(ImageFilter.GaussianBlur(3))
    img = Image.alpha_composite(img, glow3)
    # Layer 4: Crisp fill
    ImageDraw.Draw(img).text((x, y), text, fill=fill_color, font=font)
    return img
```

#### CTA Button with Shadow
```python
def draw_cta_button(img, text, cx, cy, font, button_color=(230,57,70), text_color=(255,255,255)):
    draw = ImageDraw.Draw(img)
    bbox = font.getbbox(text)
    tw, th = bbox[2] - bbox[0], bbox[3] - bbox[1]
    pad_x, pad_y = 40, 20
    x1, y1 = cx - tw//2 - pad_x, cy - th//2 - pad_y
    x2, y2 = cx + tw//2 + pad_x, cy + th//2 + pad_y
    # Shadow
    shadow = Image.new("RGBA", img.size, (0,0,0,0))
    ImageDraw.Draw(shadow).rounded_rectangle([x1+4,y1+4,x2+4,y2+4], radius=12, fill=(0,0,0,100))
    shadow = shadow.filter(ImageFilter.GaussianBlur(6))
    img = Image.alpha_composite(img.convert("RGBA"), shadow)
    # Button
    btn = Image.new("RGBA", img.size, (0,0,0,0))
    ImageDraw.Draw(btn).rounded_rectangle([x1,y1,x2,y2], radius=12, fill=(*button_color,255))
    img = Image.alpha_composite(img, btn)
    ImageDraw.Draw(img).text((cx-tw//2, cy-th//2), text, fill=text_color, font=font)
    return img
```

### The Depth Layer Stack (Compositing Order)
```
Layer 1: AI-generated background (1080x1350)
Layer 2: Dark overlay (50-70% opacity by slide type)
Layer 3: Vignette (darken edges)
Layer 4: Gradient orbs (ambient color, 1-2 per slide)
Layer 5: Glass panels (semi-transparent cards)
Layer 6: Text glow layer (blurred text behind headlines)
Layer 7: Crisp text (all text rendered clean)
Layer 8: Accent elements (lines, badges, counters, buttons)
Layer 9: Branding (watermark, handle, slide counter)
```

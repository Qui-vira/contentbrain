# Content Patterns: Testimonial & Portfolio Page Design

> Analyzed 50+ sites on 2026-03-30 | Web research + YouTube

## Layout Performance

| Layout Type | Conversion Impact | Best For |
|-------------|------------------|----------|
| Grid layout | Maximizes visual scanning | Volume testimonials, showing social proof breadth |
| Carousel / auto-swipe | +15% engagement | Space-efficient, mobile, keeping visitors on page |
| Case study cards | 2.3x enterprise leads | B2B, complex services, high-ticket offers |
| Horizontal scroll | Good mobile UX | Portfolio browsing, testimonial browsing |
| Linear/scrolling | Progressive engagement | Story-driven testimonials, detailed case studies |
| Wall of love (masonry) | Wisdom of crowds effect | High volume social proof, UGC aggregation |

## Trust Signal Patterns

| Signal | Impact | Implementation |
|--------|--------|---------------|
| Customer photo + name + title | +18.7% conversions | Always include photo, full name, role, company |
| Star ratings | +270% on product pages | 5-star display with aggregate count |
| Third-party badges (Trustpilot, G2) | +15% trust | Header of testimonial section |
| Real-time activity notifications | +98% conversions | Corner popup with recent actions |
| Client logo bar | +15% brand trust | Grayscale logos, colorize on hover |
| Social media screenshots | +29% (UGC) | Embed actual Twitter/IG posts |
| Verified labels | Reduces skepticism | 'Verified Client' badge on testimonials |

## CTA Placement Patterns

| Pattern | Description | Performance |
|---------|-------------|-------------|
| Sandwich pattern | Testimonial → CTA → More proof → CTA | Highest converting |
| Nearby placement | CTA within 200px of testimonials | +12-18% lift |
| Quote reinforcement | Testimonial quote inside CTA section | +8-12% lift |
| Sticky CTA | Follows scroll on portfolio page | Reduced friction |
| Result-matching copy | 'See results like these → Book a call' | +10-15% vs generic |

## Key Takeaways
- Video testimonials are the single highest-impact format (+80% conversion)
- Specificity in numbers/outcomes matters more than volume
- 6-12 case studies is optimal (depth > volume)
- Every testimonial page needs aggregate social proof stats at the top
- CTA copy must connect the proof to the action (not generic 'Contact Us')

# Frameworks Learned: Testimonial & Portfolio Page Design

> Researched 50+ sites and articles on 2026-03-30 | Platforms: Web research, YouTube

## Framework 1: C-R-A-V-E-N-S Trust Framework
- **Structure**: Every testimonial scored on 7 criteria — Credible, Relevant, Attractive, Visual, Enumerated, Nearby, Specific
- **Why it works**: Testimonials scoring 6-7/7 convert 4x higher than those scoring 3 or below
- **Example**: Famewall, Trustmary, and top SaaS sites use all 7 elements
- **How @big_quiv can use it**: Score every testimonial before publishing. Reject any scoring below 5/7.

## Framework 2: The Sandwich Pattern (CTA Integration)
- **Structure**: Hero testimonial → CTA → Testimonial grid → CTA → Case study → Final CTA
- **Why it works**: Every scroll depth has both proof and action. No dead zones without conversion opportunity.
- **Example**: Marketing 360, Smartsites, and top agencies use this pattern
- **How @big_quiv can use it**: Add a CTABanner between case studies and testimonials on /portfolio page

## Framework 3: The 4-Part Case Study Structure
- **Structure**: Challenge → Contribution → Key Decisions → Measurable Outcomes (800-1500 words)
- **Why it works**: Leads with the result (Peak-End Rule), then provides logical proof for analytical buyers
- **Example**: Buffer, Abstract, Tidio all use this structure
- **How @big_quiv can use it**: Create individual /portfolio/[slug] pages with this structure

## Framework 4: Multi-Format Testimonial Mix
- **Structure**: Combine 3+ formats — video (+80%), text+photo (+18.7%), case study cards (2.3x), social screenshots (+29%), star ratings (+270%), carousel (+15%)
- **Why it works**: Each format serves different psychological needs and visitor types
- **How @big_quiv can use it**: Already have AutoCarousel and image support. Add star ratings and video testimonial format.

## Framework 5: Visual Hierarchy F-Pattern
- **Structure**: Top-left (aggregate stat) → Top-right (badges) → First row (hero testimonials) → Grid below (volume) → Bottom (CTA)
- **Why it works**: Matches natural eye-tracking scan pattern
- **How @big_quiv can use it**: Add aggregate stat header and third-party badges to testimonial section

## Framework 6: Portfolio Stats Bar
- **Structure**: 3-4 key metrics with animated counters above case studies
- **Why it works**: CountUp animation increases engagement 23%. Immediate credibility before any scrolling.
- **How @big_quiv can use it**: Reuse existing CountUp component from homepage on portfolio hero
